## Usage ðŸ› 

- Clone this Repo

```bash
git clone https://github.com/mskian/tamil-blog.git
cd tamil-blog
```

- install DEPENDENCIES

```bash
bundle install
```

- Test the site

```bash
bundle exec jekyll serve
```

- Build the site

```bash
bundle exec jekyll build
```

- Create New Post

```bash
bundle exec jekyll post "My New Post"
```
